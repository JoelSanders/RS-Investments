Alternative Column Names

The system recognizes various column names for flexibility:

Data Type Recognized Column Names
Stock Symbol Symbol, Ticker, Stock Symbol, Stock
Company Name Company Name, Name, Company, Description
Shares Shares, Quantity, Qty, Share Count, Position
Average Cost Average Cost, Avg Cost, Cost Basis, Price Paid, Purchase Price
